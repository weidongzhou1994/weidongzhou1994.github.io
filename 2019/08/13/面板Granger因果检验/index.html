1. Dumitrescu-Hurlin 检验

1.1 Granger 因果检验的基本思想

  Granger 于 1969 年开创性地提出一种分析时间序列数据因果关系的方法，基本思想是：在控制 \(y\) 的滞后项 (过去值) 的情况下，如果 \(x\) 的滞后项仍然有助于解释 \(y\) 的当期值的变动，则认为 \(x\) 对 \(y\) 产生因果影响。    基本模型如下： \[y_t = \alpha + \sum_{k=1}^{K}\gamma_ty_{t-k} + \sum_{k=1}^{K}\beta_kx_{t-k} + \varepsilon_t, t = 1, … … ,T \tag1\]    检验的原假设是： \[H_0 : \beta_1 = … =\beta_K = 0\]    构造 F 统计量进行检验。如果 F 检验拒绝 \(H_0\)，则认为存在因果关系，即 \(x\) 是 \(y\) 的 Granger 因。 ## 1.2 Dumitrescu-Hurlin 的拓展   Dumitrescu-Hurlin 于 2012 年在此基础上进行了拓展，提供了一个检验面板数据因果关系的方法。基本模型如下： \[y_{i,t} = \alpha_i + \sum_{k=1}^{K}\gamma_{ik} y_{i,t-k} + \sum_{k=1}^{K}\beta_{ik} x_{i,t-k} + \varepsilon_{i,t} \tag2\]    其中，\(i = 1,… …,N; t = 1,… …,T\)。\(x_{i,t}\) 和 \(y_{i,t}\) 是两个平稳序列在个体 \(i\) 和时间 \(t\) 上的观测值。    原假设认为面板中的所有个体都不存在因果关系： \[H_0 = \beta_{i1} = … = \beta_{ik} = 0 \qquad \forall i = 1,\ldots,N_1 \tag3\]    备择假设为部分（不是所有的）个体存在因果关系： \[H_1:\beta {i1} = \ldots = \beta_{iK} = 0 \qquad \forall i = 1,\ldots,N_1 \\\\ \beta_{i1} \neq 0 or \ldots or \beta_{iK} \neq 0 \qquad \forall i = N_1+1,\ldots,N\]    其中，\(N_1 \in [0,N-1]\)。 ### 1.2.1 实现方法   Dumitrescu-Hurlin 提出运行包含在 (1) 式中的 N 个独立回归，执行 k 个线性假设的 F 检验来获得 Wald 统计量：\(W_i\)，最后计算 Wald 统计量的平均值：\(\overline {W} = \frac {1}{N}\sum_{i=1}^{N} W_i\)。    但是，拒绝 \(H_0\) 并不排除一些个体的非因果关系，Dumitrescu-Hurlin 使用蒙特卡罗模拟（Monte Carlo Simulation）发现 W 的渐进表现良好，可以用来检测面板因果关系。    在 Wald 统计量：\(W_i\) 是独立同分布的假设下，当 \(T \to \infty\) 和 \(N \to \infty\) 时，标准化的统计 \(\widetilde {Z}\) 服从下面的正态分布： \[\overline {Z} = \sqrt {\frac {N}{2K}}\times (\overline {W}-K) \quad \frac {d}{T,N \to \infty} \to N (0,1) \tag4\]    并且，对一个 T>5+3K 的固定的 T 维度来说，最大标准统计量 Z 服从以下正态分布： \[\widetilde {Z} = \sqrt {\frac {N}{2K}\times\frac {T-3K-5}{T-2K-3}}\times (\frac {T-3K-3}{T-3K-1}\times\overline {W}-K) \quad \\\\ \frac {d}{T,N \to \infty} \to N (0,1) \tag5\]    原假设 (3) 的检验也是基于 \(\overline {Z}\) 和 \(\widetilde {Z}\)， 如果这些值大于标准值，则拒绝原假设：\(H_0\)，认为 Granger 因果存在。 ### 1.2.2 滞后阶数的选择    选择滞后阶数 k 的一个方法是根据信息准则（AIC/BIC/HQIC），但这意味着 \(K_{max}^{3}\) 时间序列在整个滞后选择过程中被忽略掉了；另一个需要考虑的现实问题是面板数据中的横截面依赖性。为此， Dumitrescu 和 Hurlin 提出了 bootstrapped 临界值，计算步骤如下：    第一步：通过拟合模型（2），根据（4）和（5）的定义获得 \(\overline {Z}\) 和 \(\widetilde {Z}\)。    第二步：在 \(H_{0}:y_{i,t} = \alpha_{i}^{0} + \sum_{k=1}^{K}\gamma_{ik}^{0} y_{i,t-k} + \varepsilon_{i,t}\) 假设下拟合模型，得到残差矩阵：\(\widehat {\varepsilon}(T-K) \times N\)。    第三步：重新抽取矩阵 \(\hat {\varepsilon}\)，形成 \(\widehat {\varepsilon}(T-K) \times N\)。    第四步：随机抽取 \((y_{1}^{\star},\ldots,y_{K}^{\star})^{\prime}\)，令 \(y_{t}^{\star}=(y_{1,t}^{\star},y_{2,t}^{\star}, \ldots, y_{N,t}^{\star})\), 其中 \(k\) 为可重复的连续时间序列。    第五步：根据 \(k\) 期随机抽取，构建重抽样序列:    第六步：拟合模型 \(y_{i,t}^{\star} = \alpha_{i}^{b}+\sum_{k=1}^{K}\gamma_{ik}^{b} y_{i,t-k}^{\star} + \sum_{k=1}^{K}\beta_{ik}^{b} x_{i,t-k} + \varepsilon_{i,t}\)，计算 \(\overline {Z}^{b}\) 和 \(\widetilde {Z}^{b}\)。    第七步：重复步骤 3-6 ，根据 \(\overline {Z}^{b}\) 和 \(\widetilde {Z}^{b}\) 的分布，计算 P 值、和 \(\overline {Z}^{b}\) 和 \(\widetilde {Z}^{b}\) 的标准值。 # 2.
